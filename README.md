# Веди Landing

Статический production-проект лендинга, подготовленный для деплоя на Timeweb Cloud.

## Структура

- `index.html` — входная точка в корне репозитория
- `css/` — стили и локальные шрифты
- `js/` — основной JS-бандл
- `assets/` — изображения и файлы шрифтов

## Локальный запуск

Можно открыть напрямую как статический файл:

```bash
open index.html
```

Или проверить через простой сервер:

```bash
python3 -m http.server 8080
```

После запуска откройте:

```text
http://localhost:8080
```

## Деплой на Timeweb Cloud

Загрузите содержимое этого репозитория в корень статического сайта.

## Примечание

CTA-кнопки в исходном лендинге уже были с `href="#"`. Это поведение сохранено без изменений.
